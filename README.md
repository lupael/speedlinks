## Getting Started

### Option A – open directly
Open `index.html` in any modern browser. No server needed.

### Option B – local dev server (with live reload)
```bash
npm install
npm run dev
```

### Option C – serve with npx
```bash
npx serve . -p 3000
```
Then visit http://localhost:3000
